# SSH Allowed Signers

This file contains the SSH public keys trusted for signing tags in this repository.

## Format

```text
principal key-type public-key-data comment
```

## Current Trusted Keys

- **<mwatkins@linuxfoundation.org>** - Macbook Air M2 signing key
- **<mwatkins@linuxfoundation.org>** - Mac mini signing key

## Verification

To verify tags using these keys:

```bash
# Configure Git to use this file
git config gpg.ssh.allowedSignersFile .ssh-allowed-signers

# Verify a tag
git verify-tag v0.1.0
```
